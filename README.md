# Health-Care-Insurance-Fraud-Analysis---Power-BI
## *Project Overview:*
This project analyzes healthcare insurance claims data to identify fraud patterns, abnormal reimbursement trends, and high-risk beneficiary segments using Power BI.
The dashboard focuses on inpatient and outpatient reimbursements, provider-level patterns, demographic segmentation, and hospital duration impact on claims.

## *Key Insights:*
- Total Inpatient Reimbursement: 408M
- Total Outpatient Reimbursement: 148M
- Inpatient Beneficiaries: 31K
- Outpatient Beneficiaries: 134K
- Reimbursement trends analyzed by age group and provider
- Beneficiary distribution by state, race, and gender
- Positive relationship observed between hospital duration and claim amount
- Identified potential fraud-prone reimbursement patterns
